Bomb manual

-- Counterspell --

The instrunctions below should be considered in top-down order, calculator use is allowed.

1. Cables:

If there are 3:

  • If there is a yellow cable cut the 2nd
  • If there is a red cable cut the 1st
  • If there is a black cable cut it (there can be multiple)
  • Else cut the 3rd

If there are 4:

  • If there is a green cable cut the 1st
  • If there is a black cable cut the 2nd
  • If there is a blue cable cut 4th
  • Else cut the 3rd

If there are 5:

  • If there is a red cable cut the 1st
  • If there is a green cable cut the 2nd
  • If there is a blue cable cut the 3rd
  • If there is a yellow cable cut 4th
  • If there is a black cable cut 5th

2. Binary:

There are 2 rows of lights, which are either lit or unlit

  • The 1st row is a binary representaion of the 1st numer
  • The 2nd is a binary represantation of the 2nd numer

Decode the numers and multiply tchem to get the answer then press the buton that contains the right value

Decoding a binary numer:

  • Decoding is done from right to left
  • It is based on adding the corresponding value of 2^x if the light is on

Example:

11101 in binary = 1 * 2^0 + 0 * 2^1 + 1 * 2^2 + 1 * 2^3 + 1 * 2^4 So this numer is 29
